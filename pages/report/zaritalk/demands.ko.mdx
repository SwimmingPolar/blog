# 요구사항

## 임대정보 입력 페이지

### 1. 임대 유형 선택

- [x] 임대 유형은 `월세, 전세`입니다.
- [x] 임대 유형이 선택 됐을 때 `임대 비용 입력 인풋창`, `금액 확인하기 버튼`이 나타납니다.

### 2. 임대 비용 입력

1. 임대 유형에 따라 입력 받아야되는 화면과 데이터가 다릅니다.
   - [x] 월세 ⇒ `보증금, 월 임대료, 월 관리비, 임대료 납부일`
   - [x] 전세 ⇒ `보증금, 월 관리비, 임대료 납부일`
2. 납부하는 관리비가 없다면 체크박스를 통해서 `input`을 `disabled`를 할 수 있습니다.
   - [x] 월세 ⇒ 월 관리비 `disabled` 처리하고 **0원**으로 표기합니다.
   - [x] 전세 ⇒ 월 관리비, 임대료 납부일 `disabled` 처리하고 **0원**으로 표기합니다. 납부일은 **빈칸으로 표기합니다.**
3. - [x] 임대료 납부일은 `1 ~ 31일`까지 입력을 받을 수 있습니다.
4. - [x] 입력 받아야 되는 데이터가 하나라도 비워있거나, 받아야되는 데이터 형태가 아니면 경고 팝업을 보여주어야 됩니다.
5. - [x] `금액 확인하기 버튼`을 눌렀을 때, 모든 데이터를 체크한 후 거주 정보 입력 페이지로 넘어갑니다.

## 거주 정보 입력 페이지

### 1. 월세 환급금 예상

1. 월세 환급금은 다음과 같은 공식으로 계산됩니다.
   1. - [ ] 전세: 환급금이 없습니다.
   2. 월세:
      1. - [ ] 관리비를 제외한 `월 임대료`로만 계산합니다.
      2. - [ ] 월세 임대료는 2022년까지는 `12%`, 2023년부터는 `17%`가 환급됩니다.
      3. - [ ] 1년에 인정되는 최대 금액은 `750만원`입니다.
      4. - [ ] 현재 월로부터 최근 `5년치`를 계산하여 보여 줍니다.
      5. - [ ] 금액은 `소수점 첫째자리`에서 반올림합니다.
      6. `예시`
         1. - [ ] 현재 날짜가 2023년 2월이면서 월세 50만원일 때: 50*0.17*2 + 50*0.12*58 = 365만원
         2. - [ ] 현재 날짜가 2023년 2월이면서 월세 100만원일 때: 62.5*0.17*3 + 62.5*0.12*57 = 459만원
         3. - [ ] 현재 날짜가 2023년 2월이면서 전세일 때: 0만원
         4. - [ ] 현재 날짜가 2022년 2월이면서 월세 50만원일 때 : 50*0.12*60 = 360만원
         5. - [ ] 현재 날짜가 2022년 3월이면서 월세 100만원일 때: 62.5*0.12*60 = 450만원
         6. - [ ] 현재 날짜가 2022년 2월이면서 전세일 때: 0만원
2. - [ ] `임대비용 수정하기`를 눌렀을 때 수정하기 페이지로 이동되며, 입력했던 임대 유형과 임대 비용 정보들이 유지되어야 합니다.

### 2. 거주 정보 입력

1. - [ ] 거주 건물 입력은 최대 `30자` 입니다.
2. - [ ] 호실은 최대 `20자` 입니다.
3. - [ ] 세입자 이름은 최대 `20자` 입니다.
4. - [ ] 거주 정보가 모두 입력 됐을 때, 계약일 관련 인풋창이 나타납니다.

### 3. 계약일 입력

1. - [ ] `거주 건물, 호실, 세입자 이름` 3가지 데이터가 채워졌을 때 `계약 시작일, 계약 종료일` 이 나타납니다.
2. - [ ] 계약 시작일, 종료일은 `YYYY-MM-DD`형태로 입력을 받습니다.
3. - [ ] 계약일이 모두 입력 됐을 때, `임대인 휴대번호 입력 인풋창`과 `완료 버튼`이 나타납니다.

### 4. 임대인 휴대번호 입력

1. - [ ] 임대인 휴대번호는 최소 10자 ~ 최대 11자를 입력 받아야 됩니다. **중간에 하이픈(-) 제외합니다.**
         ex) 010-000-000 **X |** 010000000 **O**
2. - [ ] 입력 받아야 되는 데이터가 하나라도 비워있거나, 받아야되는 데이터 형태가 올바르지 않다면 경고 팝업을 보여줍니다.
3. - [ ] `완료 버튼`을 눌렀을 때, 모든 데이터를 체크한 후 완료 페이지로 넘어갑니다.

## 완료 페이지

### 1. 완료 화면

1. - [ ] 이전 페이지에서 받은 정보들을 이용하여 화면을 구성합니다.
2. - [ ] 주소가 1줄을 넘어가는 경우는 `…`(말줄임표)로 표기합니다.
3. - [ ] 납부 총액은 임대료와 관리비를 더합니다.
4. - [ ] `임대인 문자`와 `임대인 전화` 버튼을 클릭시 각각 문자, 전화 앱으로 연결합니다.

# 👍 심화 구현사항(선택사항)

1. - [ ] 거주 건물에 입력을 받았을 때 `주소API`를 통해서 입력받습니다.
   1. `kakao`, `naver` 등 원하는 API를 사용하시면 됩니다.
2. - [ ] 계약시작일, 계약종료일 달력을 직접 구현하거나, 라이브러리를 사용해서 구현합니다.
